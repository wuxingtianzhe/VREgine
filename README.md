# VREgine
> basic Engine of openGL and openVR, model&amp;material reader and basic shading

# 1.依赖库的环境配置
IDE使用vs2017 c++ 64位的debug/release

调用的库都在调用的库都在/VREngine/VREngineine/libAndInclude64文件夹中

# 2.程序运转起来之后，键盘输入

W -向前移动

A -向左移动

S -向后移动

D -向右移动

 "> 摄像头右转
 
< 摄像头左转

^ 摄像头上转

v 摄像头下转

# 3. model和shader

model的加载和shader的设置在config.txt中进行修改
